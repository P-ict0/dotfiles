# Dotfiles
Here are my config files. This repo was made to facilitate configuration of useful programs when changing or reinstalling an OS.

# Easy setup
Run  ```./configtools.sh --help ```  or  ```./configtools.sh -h ```  for instructions

The script can either: 
- Changes the current config files of the specified programs for the ones in the repo (also creates a backup of old configs)
- Fetch the current configs and store them in the repo

These two actions can be run in individual or multiple programs by specifying them as arguments
