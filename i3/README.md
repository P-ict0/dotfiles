# i3Config
My config file for i3 window manager using alacritty terminal <br>
(window workspaces, assignments and some bumblebee status modules are custom, might need to change them)

# Requirements for this *specific* config file
- PulseAudio
- Alacritty
- Rofi
- Feh
- Bumblebee-Status
- Xmodmap (used to change esc key to Caps Lock and speed up the rate of keyboard long press)
- Dropbox (initialized at the end)
- gtk-redshift (initialized at the end)

# Installation
Download/copy config file and replace it with current i3 config (probably at ~/.config/i3/). <br>
_Save old config file to prevent losing it_
